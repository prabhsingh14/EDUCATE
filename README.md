# EdTech Application - EDUCATE

## Overview
The **EDUCATE** is a comprehensive online learning platform designed to enhance the educational experience for students and educators. This MERN stack application provides users with a robust set of features, including course management, user authentication, progress tracking, and cloudinary and payment integration. The goal of this project is to create an engaging and effective digital environment for education.

## Features
- **User Authentication**: Secure sign-up, login, change, and reset password functionalities.
- **Course Management**: Create, update, and delete courses by instructors.
- **Progress Tracking**: Monitor student progress with detailed analytics.
- **Rating and Review System**: Allow students to rate and review courses to provide feedback.
- **Cloudinary Integration**: Seamless upload and management of course images and assets.
- **Responsive Design**: Fully responsive UI for optimal use on various devices.

## Technologies
This project is built using the following technologies:
- **Frontend**: React.js, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT (JSON Web Tokens)
- **Cloud Storage**: Cloudinary

## Installation
To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/prabhsingh14/EDUCATE.git

2. Navigate to the project directory:
   ```bash
   cd EDUCATE

3. Install the dependencies:
   ```bash
   npm install

4. Setup environmental variables
5. Start the development server

## Usage
Once the application is running, you can access it at http://localhost:3000. Sign up for a new account or log in with your credentials. Explore the courses, submit ratings and reviews, and track your learning progress.
